Nội dung sơ lược: Chương trình thử GCD

Các bước thực hiện bài:

Kết nối vào IP 192.168.43.223, PORT 2610, giao thức TCP

1. Gửi MSSV và MẬT KHẨU (chuỗi '2610') lên server để xác nhận làm bài. Nếu chấp nhận kết nối, server sẽ gửi về chuỗi 'CHAP NHAN'

2. Gửi một chuỗi 'GET' lên server. TH SERVER trả về chuỗi "HET", SERVER sẽ không nhận phản hồi nữa, ngược lại SERVER sẽ trả về 2 số A, B với format: "A B" (Cách nhau bằng 1 dấu trắng duy nhất)

3. Tính ước chung lớn nhất của A, B sau đó gửi lên SERVER dạng "ANSWER câu_trả_lời_của_bạn", nếu đúng SERVER sẽ gửi về chuỗi "DANH GIA: DUNG", ngược lại "DANH GIA: SAI" sẽ được trả về

Server sẽ tự động tính điểm, mỗi đáp án đúng sẽ được +1 điểm
